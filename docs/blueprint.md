# **App Name**: ATLAS: Your Life, Reborn

## Core Features:

- Twinskie Evolution: Adaptive avatar that reflects user's activity. Avatar appearance changes based on dominant skill categories. If the user becomes inactive, the avatar will slump, and desaturate to gray.
- Nebula Skill Tree: Dynamic, user-generated skill tree with AI-driven categorization, tagging user inputs into 5 Core Energies. Skills the user engages with will grow larger.
- Pioneer/Innovator Bonus: Recognize early adopters. First users to log a new skill are flagged as 'Pioneers,' earning an initial Innovation trait points. Algorithm to measure adoption, and later reward pioneer as others take it up. Will make calls to database to assess the tool adoption.  
- Social Hierarchy & Soul Link: Structure for teams and regions (City, State, Country) to drive cooperation and competition. The team of max 5 users linked. A streak logic to multiply the XP to foster collaboration between the users of one team. A notification is triggered for team streak breakage. A geolocation feature that marks locations (gyms, parks...) of interest and displays the dominating teams on a map
- Onboarding Flow: Selection of archetype. Gamified tutorial: An onboarding experience focused on immediate gratification, beginning with a profile pitched at level 0 stats, intended to demonstrate to users their 'pathetic' level, as an impetus towards engagement; and then following through with an auto-verified, simple and easy-to-achieve quest to further addict the users through quick-win levelup
- Radar Chart Summaries: Use Radar charts for summaries on the core attributes that players will want to enhance.

## Style Guidelines:

- Primary color: Electric purple (#BE29EC) for a sense of futuristic power.
- Background color: Dark slate gray (#2E2E2E) for a high-tech, dark mode aesthetic.
- Accent color: Lime green (#32CD32) to highlight key elements and calls to action with an eye-catching and hyper-modern flair.
- Headline font: 'Space Grotesk' (sans-serif) for a computerized, techy feel
- Body font: 'Inter' (sans-serif) for body text; will be paired with 'Space Grotesk' above.
- Neon-style icons to maintain the retro-futuristic aesthetic.  Radar chart summaries use distinct icons for Physical, Mental, Social, Practical and Creative skills
- A pannable, zoomable Nebula view based on an HTML5 canvas to display the user skill network and focus on nodes of interest.
